# Макет в Figma (wireframe Снэплот)

Прямого «подключения» ассистента к Figma нет: файл создаёте вы, ниже — готовая структура, размеры и тексты, чтобы собрать макет за 15–20 минут.

## 1. Новый файл

1. Figma → **New design file**.
2. Страницу переименуйте, например: `Снэплот · wireframe`.

## 2. Стилевые токены (опционально)

Создайте **Local variables** (или стили):

| Имя | Значение | Назначение |
|-----|----------|------------|
| `bg` | `#0A0C13` | фон как на сайте |
| `border` | `#475569` или 35% белого | обводка блоков |
| `text` | `#E2E8F0` | основной текст |
| `muted` | `#94A3B8` | подписи |
| `accent` | `#22D3EE` | акцент, CTA |

Шрифт: **Inter** или **Onest** (как на лендинге).

## 3. Сетка страницы

- Фрейм **Desktop**: **1440 × 900** (или 1280 × 800).
- Включите **Layout grid**: Columns, 12 col, margin 80, gutter 24 (можно упростить: 8 px grid).

## 4. Блоки лендинга (сверху вниз)

Сделайте **Auto layout** (вертикально, gap 24–32) из прямоугольников с обводкой **1 px dashed** и заливкой **bg + 4% белого**.

1. **Шапка (Nav)** — высота ~64 px  
   - Слева: логотип-плейсхолдер 32×32 + текст «Снэплот».  
   - По центру: поле «Поиск лотов…».  
   - Справа: иконки корзина/вход, кнопки «Войти» / «Регистрация».

2. **Hero**  
   - Бейдж: «Официальный магазин · автопроверка лотов».  
   - Заголовок: «Покупай аккаунты, подписки и AI-доступы без риска» (часть «без риска» цветом `accent`).  
   - Подзаголовок 2–3 строки про Steam, ChatGPT, эскроу.  
   - Кнопки: «Открыть каталог» (заливка accent), «Как это работает» (outline).

3. **Правый блок hero** (или второй столбец сетки)  
   - Карточка товара: превью, название, цена «1 490 ₽», кнопка «Купить».  
   - За ней полупрозрачные мелкие карточки (Discord и т.д.) для глубины.

4. **Полоса доверия** — три колонки:  
   - «98,4%» / сделок без споров  
   - «&lt; 2 мин» / средняя выдача  
   - «24/7» / поддержка  

5. **Направления / каталог** (заголовок секции + ряд карточек-плейсхолдеров 4–6 шт.).

Низ страницы: **Footer** — ссылки Правила, FAQ, AML, Контакты, копирайт.

## 5. Отдельные фреймы «экраны» (мини-wireframe)

Создайте на той же странице или на **Page 2** набор фреймов **320×568** или **375×812** (мобайл) / **1280×720** (десктоп) с подписями:

| Фрейм | Содержимое (текст + прямоугольники) |
|-------|-------------------------------------|
| `01 Главная` | как в п.4 |
| `02 Каталог` | фильтры сверху, сетка карточек 2×4 |
| `03 Лот` | галерея, заголовок, цена, кнопки |
| `04 Корзина / чекаут` | список строк, итого, оплатить |
| `05 Заказ / выдача` | шаги, поля для копирования |
| `06 Профиль` | аватар, бонусы, последние покупки |
| `07 Админ` | таблица лотов, кнопка «+ товар» |

Между фреймами стрелки **Connector** (Figma) или подписи «из каталога → лот».

## 6. Ускорение

- **Duplicate** первый экран и вычищайте лишнее для «Каталога» и «Лота» — так быстрее, чем с нуля.
- Онлайн-референс: открыть в браузере `docs/wireframe.html` и **скрин в Figma** (Paste) как подложка с пониженной прозрачностью, сверху нарисовать чистые блоки.

## 7. Экспорт для README (по желанию)

- Выделите нужные фреймы → **Export** PNG 2x.  
- Положите, например, `docs/figma-landing.png` в репозиторий и добавьте в README рядом со скрином сайта.
